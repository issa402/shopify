Path: Pokemon/services/analytics-engine/

            
Scheduled batch service. Computes card trends, finds deals of the day, reads/writes Postgres through repository classes, and feeds dashboard data consumed by the Go API.

            No HTTP server. Runs an infinite schedule loop.
            Pythonschedulepsycopg2scikit-learn

Path: Pokemon/services/scraping-service/

            
Go Playwright scraper for Facebook Marketplace and Mercari. Publishes scraped listings into RabbitMQ for downstream processing.

            Uses Playwright browser automation and AMQP publisher.
            GoPlaywrightRabbitMQ

Risks / Gaps Seen In The Scan

        

          
Two Port Collisions
Both NexusOS and PokemonTool compose files map Postgres 5432 and Redis 6379 on the host. Running both stacks at once on one machine will conflict unless one stack changes host ports or runs isolated.

          
NexusOS Compose Missing App Services
Root docker-compose.yml defines infra only. Go gateway, AI service, and web dashboard are run separately by Makefile/manual commands.

          
Gateway Merchant Endpoints Are Shells
/api/v1/orders, customers, decisions, approvals return empty or simple placeholder responses in services/gateway/main.go.

          
Pokemon Analytics Typo Bugs
analytics-engine/repositories/card_repo.py has likely typos in insert_price_point: INSER, avg_prcie, price_tgcplayer, conn.coomit().

          
AI Route Doc Drift
README says logistics restock and finance margin routes, but current router grep shows /agents/logistics/disruption and /agents/status/{merchant_id}.

          
External Infra Unknown
AWS/security group/DNS/TLS details are described in docs/scripts, but no live Terraform/CloudFormation/Kubernetes manifests were found in this scan.
